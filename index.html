Basics of Javascript

Data Types

  • Number - a list of integers
    Example:
    • 25
    • 905
    • 804.67

  • String - a list of characters
    Example:
    • Animal
    • Bird
    • Fishes

  • Boolean - it represents the value of the data
    Example:
    • True
    • False

  • Undefined - a value hasn't been assigned
    Example:
    • Undefined

  • Null - indicates the absence of the value
    Example:
    • No Value

Variables

  • VAR - Function scope
    Working:
    • It can be redeclared
    • It can be reassigned

  • LET - Block scope
    Working:
    • It cannot be redeclared
    • It can be reassigned

  • CONST - Block scope
    Working:
    • It cannot be redeclared
    • It cannot be reassigned

Synchronous & Asynchronous

Synchronous:
- Executes tasks one after another,
- Blocking the execution until a task is complete,
- Simple and easy to understand but can lead to performance issues with long-running tasks.

Asynchronous:
-Allows tasks to run independently,
-Enabling the program to handle other operations while waiting for certain tasks to finish.
